# Workshop DB 03 - CTEs and window functions

## Configuration

```
cp config/{database.yml.example,database.yml}
```


## Installation

```
bundle install
bundle exec rake db:create db:migrate db:seed
```


## Usage

Starting a server:

```
foreman start # app will start at http://localhost:9292
```

Starting a console:

```
./bin/irb
```
